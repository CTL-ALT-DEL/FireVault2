# Deploy FireVault 0.73.0

Deploy the contents of this ZIP to the repository root without changing the GitHub Pages address. Existing FireVault data uses the same local-storage key and database format.

After deployment:

1. Open the existing FireVault Home Screen app.
2. Allow the built-in version check to find Build 0.73.0.
3. Use **Update Now** when offered, or open **Settings → Data, Sync & Support → App Updates** and choose **Check for Updates**.
4. Confirm the displayed build is 0.73.0.

Do not delete and reinstall the Home Screen app as an update method. Deleting the installed PWA can remove device-only data and automatic snapshots. Download an external backup first whenever a reinstall is unavoidable.
