# Clean Install — Build 0.93.0

1. Export a FireVault backup from Settings before replacing hosted files.
2. Replace the deployed application files with the contents of this ZIP.
3. Open FireVault while online once so the service worker can cache Build 0.93.0.
4. Tap **Update Now** if FireVault shows an update banner.
5. Close and reopen the installed PWA to confirm the splash screen, navigation states, and form safeguards.
6. Use Repair App Files only if the installed PWA remains on an older cached build.

Existing account data and IndexedDB media are not removed by this update.
