# FireVault 0.76.0 deployment

Deploy the contents of this ZIP over the existing GitHub Pages repository. Do not delete or reinstall the Home Screen PWA.

After GitHub Pages finishes publishing, open FireVault and use Settings → Data → App Updates → Check for Updates, or reload the existing Home Screen app. The service worker uses the 0.76.0 cache name and removes older FireVault application caches without deleting the local customer vault.
