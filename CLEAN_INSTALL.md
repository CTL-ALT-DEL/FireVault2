# Clean Install — Build 0.89.0

1. Extract the ZIP archive.
2. Upload the complete contents to the FireVault hosting location, replacing the previous build files.
3. Open FireVault while online once so the updated application shell and sample preview photo can be cached.
4. Open Settings → Reports & Communication → Photo Overlays.
5. Confirm the smaller real-photo preview, Quick Setup presets, Content, Layout, and Branding sections.
6. Change a preset or overlay field and confirm the preview updates.
7. Export a test overlay photo and compare it with the preview.
8. If the previous Photo Overlay page remains visible, close every FireVault window and reopen the installed app or browser page.
9. Existing accounts, photos, overlay settings, and the FireVault storage key remain preserved.
