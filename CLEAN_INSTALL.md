# FireVault Build 0.64.0 — Clean Install

1. Extract the ZIP so `index.html`, `src`, `assets`, and `manifest.json` remain at the repository root.
2. Replace the previous FireVault files in the GitHub repository.
3. Commit and push the replacement files.
4. Wait for GitHub Pages deployment to finish.
5. Confirm the build badge reads `0.64.0`.
6. Existing local FireVault data will migrate automatically. Export a backup before replacing files as a precaution.
