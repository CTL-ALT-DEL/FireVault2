# Clean Install — Build 0.87.11

1. Extract the ZIP archive.
2. Upload the complete contents to the FireVault hosting location, replacing the previous build files.
3. Open FireVault while online once so the updated application shell can be cached.
4. Open **Settings → Data & Backup → WebDAV Backup** to confirm the restored entry.
5. If the older Settings menu remains visible, close every FireVault window and reopen the installed app or browser page.
6. Existing account data and saved WebDAV configuration remain under the same FireVault storage key.
