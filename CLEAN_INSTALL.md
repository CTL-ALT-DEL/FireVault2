# FireVault Build 0.65.1 — Clean Install

1. Export a current FireVault backup before replacing files.
2. Replace the deployed root files with the contents of this ZIP.
3. Wait for GitHub Pages deployment to complete.
4. If an older build remains cached, remove the Home Screen app and clear the FireVault website data in Safari.
5. Open FireVault in Safari and confirm the build badge reads `0.65.1`.
6. Open Settings → Customer Import and test coordinate calculation with a small CSV before importing the full customer file.
