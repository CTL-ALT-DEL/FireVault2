# Clean Install — Build 0.88.0

1. Extract the ZIP archive.
2. Upload the complete contents to the FireVault hosting location, replacing the previous build files.
3. Open FireVault while online once so the updated application shell can be cached.
4. Open Settings and confirm the sticky search bar, status cards, and grouped Settings sections.
5. Confirm WebDAV under Data & Backup, Demo Mode under Testing, and Google Plus Codes under Maps & GPS.
6. If the previous Settings design remains visible, close every FireVault window and reopen the installed app or browser page.
7. Existing account data and saved configuration remain under the same FireVault storage key.
