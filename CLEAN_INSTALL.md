# FireVault Build 0.79.0

Deploy these files over the existing FireVault GitHub Pages repository. Do not delete the installed Home Screen app and do not clear Safari website data. On first startup, FireVault creates an automatic pre-migration snapshot, upgrades the vault to security schema 4, preserves the existing `firevault_vault_build_030` key, and records the migration in the local audit history. This build does not add signup or login yet.
