# Clean Install — Build 0.87.4

1. Back up FireVault data before replacing application files.
2. Replace the deployed files with the complete contents of this package.
3. Commit and push the updated files to GitHub Pages.
4. On iPhone, close FireVault completely and reopen it.
5. When prompted, install the available update. If the old build remains cached, remove the Home Screen PWA and add it again.

The existing FireVault storage key is unchanged, so customer records remain compatible.
