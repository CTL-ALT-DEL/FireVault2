# FireVault App Profile

FireVault Build 0.95.7 uses `src/app-profile.js` as the active configuration for identity, terminology, modules, the record data model, and field workflow presets.

The FireVault profile remains optimized for fire alarm technicians. Shared screens consume the profile so future AppForge products can change terminology, fields, detail sections, photo categories, action surfaces, and Quick Photo behavior without rewriting the shared application core.

The active profile uses schema version 5 and the `fire-alarm-technician` workflow preset.
