# FireVault2

FireVault is a local-first field knowledge system for fire alarm technicians.

## Version 1 Goal
Create customer sites, save panel/access notes, add visit history, and export/import backups.

## What is included
- React + Vite app
- Mobile-friendly interface
- Today dashboard
- Sites list and search
- Add/Edit Site
- Site detail page
- Visit history
- Resource library
- Local browser storage
- Export/import backup
- PWA manifest

## What is not included yet
- Firebase login
- Cloud sync
- Photos
- GPS matching
- Background location
- AI search

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Development rule
Build one feature, test it, commit it, then move to the next feature.
