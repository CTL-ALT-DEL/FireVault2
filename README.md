# FireVault Build 0.94.8

## Technician Overlay Alignment

- Repairs Technician Overlay Template text wrapping so names, email addresses, and other longer values remain on one clean line.
- Adds one group-level alignment control: Flush Left, Center, or Flush Right.
- Applies the selected group alignment when Technician Info is added to Photo Overlay.
- Extends the live/export overlay renderer to support centered fields.
- Technician Info lines are marked no-wrap and automatically reduce or fit their text to the available photo width.
- Simplifies Technician Template field rows by removing per-field alignment controls; order and line breaks remain editable.
- Existing technician profile information, template fields, accounts, photos, WebDAV, Demo Mode, IndexedDB media, and storage keys remain compatible.
