# FireVault Build 0.94.9

## Tech Info Overlay Field

- Photo Overlay now shows **Tech Info** as one field instead of exposing Technician, Company, Phone, Email, and License separately.
- The contents and internal line arrangement of Tech Info continue to come from **Settings → Profile → Technician Overlay Template**.
- Tech Info can be moved up or down, removed, and aligned Left, Center, or Right like the other Photo Overlay fields.
- The Profile template remains the single place to choose, order, and group the individual technician details.
- Existing Photo Overlay layouts containing individual technician fields automatically migrate into one Tech Info field.
- The live preview and exported image expand Tech Info into the saved Profile template without changing the Photo Overlay field list.
- Existing accounts, photos, technician profile information, WebDAV, Demo Mode, IndexedDB media, and the `firevault_vault_build_030` storage key remain compatible.
