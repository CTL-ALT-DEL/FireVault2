# FireVault Build 0.68.0

Major UX redesign focused on the three most-used navigation areas.

- Account View reorganized around service actions, account snapshot, work history, resources, and location.
- Settings consolidated into five purpose-based folders while preserving every existing setting.
- FireVault Academy simplified into a calm, continuous-reading Help experience.
- Existing customer data, CSV import, coordinate calculation, Nearby Sites, service visits, Team Sync preparation, and backup workflows remain intact.


## Build 0.68.0
