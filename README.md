# FireVault Build 0.45.0

Build 0.45.0 is a stability checkpoint and dashboard status polish build. It does not add a large new module; it keeps the current feature set intact and makes the main screen better at showing update safety status.

## Changes
- Visible app version advanced to 0.45.0.
- Added a compact dashboard Stability Check card.
- Stability Check shows sites, visits, GPS-saved sites, and last backup export visibility.
- Kept the dashboard focused on field actions instead of duplicating lower-menu items.
- Tightened the dashboard hero/build badge layout.
- Preserved Settings readability polish, Backup Center, Report Center, Site Detail polish, Attention Queue, Deficiency Center, Inspection Checklist, GPS/Nearby, haptics, refined 3D controls, and the Loading FireVault boot watchdog.

## Install
Upload the ZIP contents to the repository root. Do not upload the containing folder itself.
