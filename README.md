# FireVault Build 0.42.6

Unified 3D button design and roomier Settings submenu layout.

## Updates
- Replaced mixed colored/glow shadow styling with one consistent raised 3D button system.
- Added tactile pressed/down states for buttons, tabs, chips, task filters, quick events, and bottom navigation.
- Kept color accents for meaning only, while using neutral shadows for depth.
- Changed the Settings submenu selector into a full-width responsive grid so tabs use the available space instead of feeling cramped.
- Preserved the 0.42.4 Loading FireVault fix and 0.42.5 bottom-menu position.
- Preserved GPS/Nearby Sites, Task Center, Visit Log, service follow-ups, haptics, and map tools.

## Install
Use the clean-root ZIP contents in the GitHub Pages project root. Do not nest build folders.
