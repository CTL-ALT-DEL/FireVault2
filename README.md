# FireVault Build 0.43.1

Site Documents Vault starter build for storing customer-specific manuals, permit numbers, inspection forms, account references, drawings, and useful links.

## Updates
- Advanced visible version to 0.43.1.
- Added a per-site Documents / Links Vault.
- Added document/link types for panel manuals, permits, inspection forms, monitoring accounts, contracts, photo sets, drawings, code references, and other references.
- Added title, reference/account/permit number, URL/link, date/revision, and notes fields.
- Added quick Open Link and Copy actions for document references.
- Added a Documents / Links tile and summary card on Site Detail.
- Reports now include a Documents / Links section.
- Diagnostics now includes total saved documents.
- Preserved the 0.42.4 Loading FireVault fix, refined 3D controls, GPS/Nearby Sites, Task Center, Visit Log, Equipment Vault, Contacts & Access Vault, haptics, and map tools.

## Install
Copy the contents of this ZIP into the GitHub Pages project root. Do not copy the folder itself into the repo.
