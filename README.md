# FireVault Build 0.75.0 — Technician UX Refresh

Build 0.75.0 is a usability and visual-cohesion milestone. It does not replace the existing FireVault data model. It standardizes page spacing, touch targets, card hierarchy, account-detail actions, tabs, search/filter controls, and global navigation.

## Highlights

- Faster, larger Call, Route, Note, and Visit actions with consistent line icons and supporting labels.
- A clearer Account Detail identity header with account, address, health, category, task, and deficiency context.
- Improved Accounts directory density, search visibility, row scanning, and active feedback.
- Larger, better-spaced Settings tabs and consistent settings cards.
- Unified page margins, surfaces, typography, focus states, and responsive behavior.
- Existing Demo Mode, backups, updates, categories, imports, and storage key remain unchanged.
