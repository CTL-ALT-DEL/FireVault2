# FireVault Build 0.43.0

Contacts & Access Vault starter build for per-site customer, access, and after-hours details.

## Updates
- Advanced visible version to 0.43.0.
- Added a per-site Contacts & Access Vault.
- Added contact records for customer, property manager, maintenance, security, monitoring, emergency, access, and other contacts.
- Added fields for phone, email, role/title, after-hours contact flag, access notes, and general notes.
- Added quick Call and Email actions from contact cards where the device/browser allows it.
- Added a Contacts & Access summary card on Site Detail.
- Reports now include a Contacts / Access section.
- Diagnostics now includes total saved contacts.
- Preserved the 0.42.4 Loading FireVault fix, refined 3D controls, GPS/Nearby Sites, Task Center, Visit Log, Equipment Vault, haptics, and map tools.

## Install
Copy the contents of this ZIP into the GitHub Pages project root. Do not copy the folder itself into the repo.
