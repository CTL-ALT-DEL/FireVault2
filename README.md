# FireVault Build 0.42.3

iPhone shell, Settings readability, and GPS visibility polish build.

## Updates
- Tuned the iPhone home-screen shell and bottom dock spacing to reduce the blank area below the menu icons.
- Made Settings fonts slightly larger and relaxed the ultra-compact spacing.
- Kept the minimalist pill-tab Settings design, but moved GPS forward so it is easier to find.
- Added a GPS / Nearby status strip to the Sites page with direct Scan Nearby access.
- Kept Nearby Sites detection, GPS capture, Apple/Google Maps, Visit Log, Task Center filters, service follow-ups, and haptic-ready controls.

## Install
Use the clean-root ZIP contents in the GitHub Pages project root. Do not nest build folders.
