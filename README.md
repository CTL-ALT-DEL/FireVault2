# FireVault Build 0.48.0

Build 0.48.0 restores the Apple-inspired home concept proportions and removes the extra app chrome that made the layout feel wrong.

## What changed

- Visible app version advanced to 0.48.0.
- Rebuilt from the better-looking 0.47.8 Apple-inspired concept baseline, not the squished 0.47.9 overcorrection.
- Removed the extra global FireVault header from the Home screen.
- Removed the extra bottom app nav from the Home screen.
- Kept the concept-style in-page top row with menu, FireVault brand, and release/history button.
- Restored larger, cleaner concept #2 spacing and card proportions.
- Kept customer search, Nearby Accounts, Recent Accounts, stats, floating add, Modules, and all 0.47.x features.

Note: the Safari address bar and Safari bottom toolbar are browser controls. They disappear when FireVault is installed/opened from the iPhone Home Screen as a PWA.

Build 0.48.0 restore apple concept home and remove extra chrome
