# FireVault Build 0.44.9

Build 0.44.9 is a Backup Center / data-safety polish pass. It keeps the current feature set intact and improves the Settings → Backup workflow so the app is safer to update between GitHub Pages deployments.

## Changes
- Visible app version advanced to 0.44.9.
- Redesigned Settings → Backup into a cleaner Backup Center.
- Added a compact data safety snapshot with counts for sites, visits, open tasks, and GPS-saved sites.
- Added Copy Summary for a quick text record of the current vault state.
- Exported backup filenames now include the current date and build number.
- Added last-export visibility on the Backup screen.
- Separated Backup actions into Export, Import / Restore, and Danger Zone sections.
- Preserved Settings readability polish, Report Center, Site Detail polish, Attention Queue, Deficiency Center, Inspection Checklist, GPS/Nearby, haptics, refined 3D controls, and the Loading FireVault boot watchdog.

## Install
Upload the ZIP contents to the repository root. Do not upload the containing folder itself.
