
## Build 0.95.0 — Quick Account Photo

- Adds a fourth **Photo** button to the Nearby, Search, Account, and Settings bottom navigation.
- Uses the current account, selected Nearby account, or last photo account.
- Opens an account picker when no account is available.
- Resizes camera images to a maximum 2048-pixel side before storage.
- Shows a live preview with the selected account's Photo Overlay.
- Confirms the account before saving and allows account reassignment.
- Remembers the last photo category.
- Saves photos through the IndexedDB media-storage protection layer.

- Customer records, IndexedDB media, WebDAV, Demo Mode, and the existing storage key.
