# FireVault Build 0.94.7

## Technician Profile Completion

- Adds a technician profile-photo uploader under Settings → Profile.
- Center-crops and resizes uploaded photos to 384 × 384 JPEG to keep app storage manageable.
- Organizes Technician Photo, Identity, Contact, and Technician Overlay Template into collapsible sections.
- Incomplete required sections open automatically.
- Completed sections collapse by default and remain closed until manually reopened.
- Manual open/closed choices for completed sections are remembered on the device.
- Identity and Contact sections have their own save controls.
- Existing technician information, overlay templates, accounts, photos, WebDAV, Demo Mode, and storage keys remain compatible.
