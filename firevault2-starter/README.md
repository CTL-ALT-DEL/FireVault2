# FireVault

FireVault is a field knowledge system for fire alarm technicians.

## Version 1 goal

Create and reopen customer site memory:

- Site/customer records
- Address and GPS coordinates
- Panel make/model
- FACP and annunciator locations
- Access notes
- Monitoring/contact notes
- Visit history
- Basic resource library
- Local backup export/import

## Run locally

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Development rule

Build one feature, test it, commit it. Do not rewrite the whole app to fix one feature.

## Roadmap

2. Visit start/stop timer
3. Photo attachments
